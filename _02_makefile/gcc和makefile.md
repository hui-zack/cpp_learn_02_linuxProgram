## GCC
**gcc编译四步骤**
    
                 预处理           编译            汇编             链接
        hello.c  ----> hello.i  ---->  hello.s  ----> hello.o  <----> a.out
                gcc -E          gcc -S          gcc -c 

            -o          // 自定义输出文件名称            

            预处理: 
                展开宏, 头文件
                替换条件编译
                删除注释, 空行和空白
            编译:
                检查语法规范, 将代码转换为汇编语言
            汇编:
                将汇编语言转化为机器指令
            
            链接: 
                数据段合并, 数据地址回填

**gcc常用参数**
[编译时指定头文件]
    git hello.cpp -o a.out -I /home/hui/hello.h

    -v                          // 参看版本
    -I[hedersDir]               // 指定头文件目录, 注意I和目录中间没有空格
    -c                          // 只编译生成.o文件, 不金i选哪个链接
    -g                          // 包含调试信息
    -On n=[0~3]                 // 编译优化, n为3时优化最多
    -Wall                       // 提示更多警告信息
